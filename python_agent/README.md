# StudyMate AI - Python ADK Microservice

Microservice powered by **Google ADK (`google-genai`)** and **Gemini 2.5 Flash / Pro** managing multi-agent workflows.

## Agents
- **CoordinatorAgent**
- **PlannerAgent**
- **TutorAgent**
- **ResearchAgent** (with YouTube video & transcript search)
- **QuizAgent**
- **ProgressAgent**
